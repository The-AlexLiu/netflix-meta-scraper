## ğŸš€ Quick Start

### Prerequisites

- Python 3.9+
- Node.js 16+ (for frontend)

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/The-AlexLiu/netflix-meta-scraper.git
    cd netflix-meta-scraper
    ```

2.  **Install Backend Dependencies**

    ```bash
    pip install -r requirements.txt
    playwright install chromium
    ```

3.  **Install Frontend Dependencies**
    ```bash
    cd frontend
    npm install
    cd ..
    ```

### Usage

#### Option 1: Premium Web Interface (Recommended)

1.  **Start the Backend API**

    ```bash
    python3 app.py
    ```

    _Server runs at http://localhost:8000_

2.  **Start the Frontend UI** (in a new terminal)

    ```bash
    cd frontend
    npm run dev
    ```

    _UI opens at http://localhost:5173_

3.  **Open Browser**: Go to `http://localhost:5173`.
4.  **Configure & Run**: Set your start/end dates and click the lightning **INITIATE** button.
5.  **Export**: Once finished, click **"Download All"** to get your zipped data.

#### Option 2: CLI Mode (Advanced)

You can also run the scraper directly from the command line:

```bash
python3 netflix_scraper.py --start 2026/02/09 --end 2026/02/15
```
